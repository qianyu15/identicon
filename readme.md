# Identicon Generator

A simple identicon generator made with HTML, CSS, and JavaScript.

Generate a unique pixel-style avatar from any text.

## Features

- Generate identicons from text
- Same text creates the same icon
- Download as PNG
- No dependencies

## Usage

Open `index.html` in your browser and enter any text.

## Built With

- HTML
- CSS
- JavaScript
- Canvas API
